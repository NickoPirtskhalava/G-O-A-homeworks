CSS საზომი ერთეულები
ერთეული აღწერა მაგალითი
px ფიქსირებული პიქსელი — არ იცვლება ეკრანის ზომის მიხედვით width: 100px;
% პროცენტი — დამოკიდებულია მშობელი ელემენტის ზომაზე width: 50%;
rem ფესვის (root) ელემენტის ფონტის ზომაზე დამოკიდებული font-size: 1.5rem;
em მშობელი ელემენტის ფონტის ზომაზე დამოკიდებული padding: 2em;
vh ვიუპორტის სიმაღლის 1% — ეკრანის სიმაღლეზე დამოკიდებული height: 100vh;
vw ვიუპორტის სიგანის 1% — ეკრანის სიგანეზე დამოკიდებული width: 100vw;
